    Pertanyaan Umum

    PC Game Pass adalah langganan bulanan yang memberi Anda akses ke ratusan game PC berkualitas tinggi di perangkat PC Windows. Cara terbaik untuk menikmati PC Game Pass adalah di aplikasi Xbox untuk Windows, tersedia untuk diunduh dari Microsoft Store atau xbox.com.

    Kunjungi wilayah Xbox untuk detail tentang ketersediaan pasar.

    Anggota Game Pass dapat menikmati akses ke game berkualitas tinggi di katalog PC hingga keanggotaan dibatalkan/kedaluwarsa, atau game meninggalkan katalog Game Pass.

    Fitur penagihan berulang diaktifkan secara default, artinya Anda akan otomatis membayar periode langganan baru saat periode saat ini berakhir.

    Anda dapat menonaktifkan dan mengaktifkan kembali tagihan berulang dari konsol Anda atau melalui akun Microsoft. Pelajari selengkapnya tentang menghentikan penagihan berulang di Dukungan Microsoft.
